# yukTravelling! | Travel Landing Page

A travel agency landing page called **yukTravelling! G03**, built with pure HTML & CSS as our first independent web development project (Team G03). The site showcases travel activities, booking features, and service highlights for exploring Indonesia's natural wonders.

---

## Preview

> A full travel landing page with a hero section, activity cards, booking bar, service highlights, and a footer with payment methods.

---

## Features

- **Fixed Navigation Bar** : gradient blue header with smooth hover effects
- **Hero Section** ": full-screen background image with headline and tagline
- **Activity Cards** : hover-animated cards for Rafting, Hiking, and Camping
- **Booking Bar** : inline search input with an "Order Now" button
- **Why Us Section** : 4-column grid highlighting key benefits (payments, discounts, points, guarantee)
- **Footer with Payment Info** : links and bank logo images (BCA, Mandiri, BRI, BNI)
- **Responsive Design** : adapts to tablet and mobile screens with a hamburger menu

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/bellatrijuliana/project-travel-web.git
   ```
2. Navigate to the project folder:
   ```bash
   cd project-travel-web
   ```
3. Open `index.html` in your browser.

> No installation or server required — just open it directly in the browser!
